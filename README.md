# AUTOBAN

<details>
  <summary>ｺｸ English version</summary>

Simple script to ban bots at Twitch chats using a text file as a source.

## How to use

### Windows

Go to [releases](https://github.com/xornotor/autoban/releases) for further instructions.

### Linux/MAC

Build from source by executing

```sh
python3 -m venv env
source env/bin/activate
pip install -e .
```

**ATENﾃﾃグ**: You may run `autoban .` and within 5 seconds you will put your cursor on the Twitch chat. PLEASE, DO NOT INTERACT WITH YOUR KEYBOARD AND MOUSE DURING THE SCRIPT HANDLING. Wait for the script to end and then you are done.

</details>
